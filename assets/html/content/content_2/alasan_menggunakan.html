<-- Alasan Menggunakan Spec Tersebut? -->

Last Update : 2022-12-09 06:57:05


Cpu

Alasan menggunakan cpu i5 11400 adalah karena di harga 2 jutaan di tahun 2021 kita sudah bisa membeli, mempunyai 6 core 12 thread i5 11400 sudah mencukupi kebutuhan saya di tambah i5 yang saya clock speed nya di 4.33Ghz ini membuat cpu saya yaa bisa lah untuk berproduktifitas seperti mengoding, main game, mengedit.

Ram

Untuk ram saya mengambil 16 saja (2x8), cukup lah untuk kebutuhan di tahun 2022, saya juga tidak banyak membuka software secara bersamaan, jadi ram saya tidak mengambil 32Gb karena tidak akan terpakai juga.

Vga

Vga saya akan menggunakan antara rx6700xt (2End) dan 3060ti (2End)). Kenapa?, harga second dari kedua vga tersebut lumayan tidak jauh dari segi harga dan performa yg di tawarkan hanya saja dari kedua vga tersebut memiliki keuntungan yang berbeda beda, sebenar nya saya lebih tertarik ke rx6700xt ketimbang rtx3060ti, alasanya simple, harga yang sedikit lebih murah, jumlah vram 12gb, dan kemampuan gaming di atas 3060ti. Walaupun rtx lebih untuk di rendering, ray tracing. Lebih kearah produktifitas.

Ssd

256Gb sudah cukup untuk menyimpan os saja dan beberapa software yang benar2 saya ingin percepat, jadi untuk penyimpan lain nya saya simpan di hdd

Psu

Psu saya menggambil Fsp Hv pro 550w karena itu sudah cukup untuk menampung semua listrik komponen pc saya, selain itu juga suda pure 85+ bronze.

Hdd

Hdd saya mengambil 2Tb karena saya memang butuh storage sebanyak itu untuk keperluan produktifitas saya, saya juga gemar mengedit video yg mana itu membutuhkan banya storage. Game yg saya main kan juga membutuhkan banyak rom sperti apex, overwatch, rencana nya mau main gta5 dan cod mw2.

Casing

Untuk casing saya mencari yang standar saja di harga 500rban sudah dapat casing atx + free 4 fan (3intake depan, 1exhaust belakang)

Cpu Cooler

Kerena budget tipis saya memilih cpu cooler sejutah umat jonsbo cr1200, di harga 120rb sudah bisa menjinakan cpu sekelas i5 gen 11 yang kata nya panas.

motherboard

Dan yang terakhir saya memilih motherboard kasta tertinggi di b560 msi yaitu msi b560 tomahawk wifi, alasan nya simple. Sudah memiliki heatsink vram yang adem, 4 slot ram support ddr5, ssd support gen 4, 3 ssd, dan yang pasti nya ada nyala2 rgb brow.
